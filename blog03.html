如何使用以下三个命令

1.git init

这个命令的作用是初始化当前目录并创建一个空的本地仓库".git"。使用"git init"命令前一定要注意当前所在目录，

2.git add

这个命令的作用是将工作区的文件添加到本地的暂存区，添加指定文件时用"git add 文件名"，使用"git add ."可以将工作区所有文件提交到暂存区。工作区的文件必须先添加到暂存区才能提交到本地仓库。

3.git commit -v

这个命令的效果是将本地暂存区的文件正式提交到本地仓库，并显示所有diff信息，每次提交都必须有添加提交信息，一般使用命令"git commit -v -m '提交信息'"。
